ビルド手順書 | VoiceChanger

実行環境

この手順は、Pythonがインストールされていることを前提としています。（Python 3.12.6 推奨）

1. VoiceChangerのgitrepositoryをクローン

以下のコマンドを使用して、リポジトリをクローンしてください。

git clone https://github.com/KD-Standup-Thumb/VoiceChanger_v1.0.0.git

2. Python仮想環境（venv）の構成

仮想環境を構成して、依存関係を管理します。

以下のコマンドをターミナルで実行してください。

python -m venv --prompt . .venv
.venv\Scripts\activate

仮想環境をアクティベート後、依存関係をインストールします。

pip install -r requirements.txt

3. APIサーバーの起動

次に、APIサーバーを起動します。以下の手順で行ってください。

    APIサーバーを起動します。

    python vch_api.py

4. Webサーバーの起動

最後に、Webサーバーを起動します。以下のコマンドをターミナルで実行してください。

streamlit run app.py

ビルド完了

これでVoiceChangerのビルドが完了です。アプリケーションを利用できます。
